# Helpdesk Client

## Overview
Helpdesk Client is a React-based web application designed to manage tickets, users, and dashboards for a helpdesk system. It provides an intuitive interface for administrators and users to interact with the system efficiently.

## Features
- **Authentication**: Secure login and registration.
- **Dashboard**: Overview of system statistics.
- **Ticket Management**: Create, view, and update tickets.
- **User Management**: Add, view, and manage users.
- **Charts and Analytics**: Visual representation of data.

## Technologies Used
- **React**: Frontend framework.
- **TypeScript**: Strongly typed JavaScript.
- **Material-UI**: UI components.
- **Axios**: HTTP client for API calls.
- **Vite**: Development environment.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/helpdesk-client.git
   ```
2. Navigate to the project directory:
   ```bash
   cd helpdesk-client
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Running the Project
Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`.
